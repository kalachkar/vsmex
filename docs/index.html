About

VS Code's extension marketplace has become a target for supply chain attacks. When Microsoft flags and removes a malicious extension, the VSIX package is no longer available for download. VSMEx continuously monitors Microsoft's malicious extensions and removed packages lists and captures the VSIX files before removal.

The dataset is described in our CODASPY 2026 paper.

Access

VSIX packages are not publicly available to prevent misuse. Access is provided to approved researchers via SURFfilesender. Email from an institutional address with your name, affiliation, and intended use.
